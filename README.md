# Simple universal web app demo

#### What's inside:

- React
- Express
- Webpack
- Babel
  - ES2015
  - JSX
- Extract css from to `.css` file


#### How to run
```
npm install

# watch and build server.js and client.js, main.css
npm run watch

# start node.js server
npm run dev

open https://localhost:3000
```
